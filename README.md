# Исследовательский проект (временные ряды)

### Выполнил: Артюхов Роман

### В рамках проекта была проверена гипотеза:
Разные модели и разные стратегии конкурентоспособны на разных участках прогнозного горизонта

### Данные:
В качестве данных использован датасет стоимости фруктов и овощей в Непале. Ссылка: https://www.kaggle.com/datasets/ramkrijal/agriculture-vegetables-fruits-time-series-prices 


### Формат репозитория

| Объект  | Описание |
| ------------- |:-------------:|
| /dataset      |папка для хранения датасета    |
| /src      | набор кода для проведения эксперимента     |
| TS_project.ipynb     | ноутбук для запуска экспериментов и подсчета результатов     |
| requirements.txt    | зависимости     |
